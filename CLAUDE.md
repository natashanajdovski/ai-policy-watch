# Claude Code — project rules

This file exists because Claude Code auto-reads `CLAUDE.md` at the repo
root on session start. The actual project conventions, architecture
rules, and tone principles for any AI agent acting in this repo live in
the portable [AGENTS.md](./AGENTS.md) file.

**Read [AGENTS.md](./AGENTS.md) first.** Everything there applies. The
highlights — don't editorialize, don't add urgency, don't commit secrets,
don't expand scope without a human decision, respect content-type
isolation, schema-is-the-contract — are non-negotiable and trip people
up fastest.

After reading AGENTS.md, also read [docs/PRODUCT_BRIEF.md](./docs/PRODUCT_BRIEF.md)
for the authorial intent, especially sections 1–5, 7, 10, and 12, before
making any non-trivial change.
